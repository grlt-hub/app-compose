---
title: when
description: "Bundles `every`, `some`, and `not` into a `{ context, fn: Boolean }` accepted by task.enabled or task.run."
---

`when` bundles `every`, `some`, and `not` into `{ context: Spot<boolean>, fn: typeof Boolean }` — accepted by `task.enabled` and `task.run`.

## when.every

Same as `every`.

```ts del={4-7} ins={13}
createTask({
  name: "_",
  run: { fn: console.log },
  enabled: {
    context: every([fetchUser, fetchOrder], (task) => task.result.id % 2 === 0),
    fn: Boolean,
  },
})

createTask({
  name: "_",
  run: { fn: console.log },
  enabled: when.every([fetchUser, fetchOrder], (task) => task.result.id % 2 === 0),
})
```
