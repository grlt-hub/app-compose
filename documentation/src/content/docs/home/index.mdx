<CardGrid>
  <Card title="High coupling" icon="puzzle">

  </Card>
  <Card title="Reuse means copy-paste" icon="add-document">
    You need the same feature, just slightly different — so you duplicate it.
  </Card>
  <Card title="Silent breaks" icon="warning">

  </Card>
  <Card title="What's running here?" icon="list-format">

  </Card>
  <Card title="No control over startup order" icon="random">

    runtime.
  </Card>
  <Card title="Performance black box" icon="magnifier">
    No idea which initialization step is slow — or whether your last change made it worse.
  </Card>
</CardGrid>

<div class="lead">
