This is a static documentation site. It has no accounts, no forms, and it sets no cookies. There are no analytics or tracking scripts in your browser.

## Serving the site

The site is served by GitHub Pages (origin) and Cloudflare (CDN). To deliver pages, they process standard request data on their side: your IP address, the requested URL, and your browser's user agent. We see this data only as aggregated statistics — for example, total visits per country. We cannot see or identify individual visitors.

How these providers handle request data is described in the [GitHub Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement) and the [Cloudflare Privacy Policy](https://www.cloudflare.com/privacypolicy/).

## The Share button in the sandbox

When you press Share in the [sandbox](/sandbox/), the code from the editor is sent to our snippet store. Three things to know:

- The snippet is stored under a random id. Anyone who has the link can read it.
- Snippets do not expire. They stay until we remove them.
- Your IP address is used for a moment to limit how often one client can share. It is not stored with the snippet.

Do not put personal data or secrets into code you share.
