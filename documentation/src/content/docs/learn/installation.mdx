App-Compose is a lightweight and standalone library with zero dependencies. TypeScript support works out of the box. No separate `@types` packages are required.

## Try App-Compose

You don’t need to install anything to play with App-Compose. Try editing the [sandbox](/sandbox/) in a new tab.

## Install the package

Use your preferred package manager.

<Tabs>
  <TabItem label="npm">
  
    ```bash
    npm install --save-exact @grlt-hub/app-compose

    ```

  </TabItem>
  <TabItem label="pnpm">

    ```bash
    pnpm add --save-exact @grlt-hub/app-compose

    ```

  </TabItem>
  <TabItem label="yarn">

    ```bash
    yarn add --exact @grlt-hub/app-compose

    ```

  </TabItem>
  <TabItem label="bun">

    ```bash
    bun add --exact @grlt-hub/app-compose

    ```

  </TabItem>
</Tabs>

## Using with AI assistants

### Cursor

1. Open chat and type `@docs`
2. Click **Add new doc**
3. Paste the URL and confirm:

```
https://app-compose.dev/llms-full.txt
```

### Claude / ChatGPT / Copilot

Paste this URL into the chat — most assistants accept a URL as context:

```
https://app-compose.dev/llms-full.txt
```

Use [`llms-small.txt`](https://app-compose.dev/llms-small.txt) for models with a smaller context window (e.g. GPT-3.5, free-tier plans).
