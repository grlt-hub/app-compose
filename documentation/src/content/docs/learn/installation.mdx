App-Compose is a lightweight and standalone library with zero dependencies. TypeScript support is built in and you do not need separate `@types` packages.

## Try App-Compose

You don’t need to install anything to play with App-Compose. Try editing this sandbox!

## Install the package

Use your preferred package manager.

<Tabs>
  <TabItem label="npm">
    ```bash
    npm install --save-exact @grlt-hub/app-compose

    ```

  </TabItem>
  <TabItem label="pnpm">
    ```bash
    pnpm add --save-exact @grlt-hub/app-compose

    ```

  </TabItem>
  <TabItem label="yarn">
    ```bash
    yarn add --exact @grlt-hub/app-compose

    ```

  </TabItem>
  <TabItem label="bun">
    ```bash
    bun add --exact @grlt-hub/app-compose

    ```

  </TabItem>
</Tabs>
