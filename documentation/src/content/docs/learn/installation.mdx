App-Compose has zero dependencies. Written in TypeScript — no `@types` package needed.

## Try App-Compose

You don’t need to install anything to try App-Compose. Open the [sandbox](/sandbox/) in a new tab.

## Install the package

Use your preferred package manager.

<Tabs>
  <TabItem label="npm">
  
    ```bash
    npm install --save-exact @app-compose/core

    ```

  </TabItem>
  <TabItem label="pnpm">

    ```bash
    pnpm add --save-exact @app-compose/core

    ```

  </TabItem>
  <TabItem label="yarn">

    ```bash
    yarn add --exact @app-compose/core

    ```

  </TabItem>
  <TabItem label="bun">

    ```bash
    bun add --exact @app-compose/core

    ```

  </TabItem>
</Tabs>
