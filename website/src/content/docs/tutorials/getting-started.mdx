## Installation

Note: Effector version 23 is a peerDependency of the library.

<Tabs>
  <TabItem label="npm">
  ```sh [npm]
  npm i effector @grlt-hub/app-compose
  # or
  npm i @grlt-hub/app-compose
  ```
  </TabItem>
  <TabItem label="yarn">
  ```sh [yarn]
  yarn add effector @grlt-hub/app-compose
  # or
  yarn add @grlt-hub/app-compose
  ```
  </TabItem>
   <TabItem label="pnpm">
  ```sh [pnpm]
  pnpm add effector @grlt-hub/app-compose
  # or
  pnpm add @grlt-hub/app-compose
  ```
  </TabItem>
</Tabs>
